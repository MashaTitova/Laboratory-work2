# Laboratory-work2
Создание программы для нахождения определенного члена арифметической прогрессии, суммы членов и визуализации этой прогрессии.
Приложение с графическим интерфейсом Windows Form, написанное на C# для:
- Нахождения n-ого члена арифметической прогрессии
- Вычисления суммы членов арифметической прогрессии
- Визуализации прогрессии

## Как запустить
### Через Visual Studio 
1.Склонируйте репозиторий
2. Откройте файл `Lab_2.var26.sln`
3. Запустите программу
#### Как склонировать репозиторий
1.Скопируйте ссылку на репозиторий с GitHub.  
2.В терминале введите команду git clone <ссылка на репозиторий>. 
3.После выполнения команды в текущей папке появится папка с тем же именем, что и сам репозиторий. Если нужно указать другую папку, в конце команды для клонирования нужно указать её название: git clone <ссылка на репозиторий> <папка>.
##### Пример использования
1.Пользователь вводит занчения первого элемента последовательности, разности элементов и количества элементов в соответствующие поля.
2.При неверном вводе подсчет результатов невозможен
3.При нажатии кнопки на листе SequnseGenerator происходит вывод арифметической прогрессии.
4.При нажатии кнопки на листе ProgressionSum происходит вывод суммы прогрессии.
5.При нажатии кнопки на листе ProgressionVisualizing происходит визуализация арифметической прогрессии.

###### Особенности программы
1. Полная валидация ввода (проверка на число, пустой ввод)
2. Графический интерфейс
3. Визуальное отображение результата


###### Требования 
1. .NET 6.0 или новее
2. Visual Studio 2022 (или любая среда с поддержкой C#)
