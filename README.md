# Laboratory-work2
Создание программы для нахождения определенного члена арифметической прогрессии, суммы членов и визуализации этой прогрессии.
Приложение с графическим интерфейсом Windows Form, написанное на C# для:
- Нахождения n-ого члена арифметической прогрессии
- Вычисления суммы членов арифметической прогрессии
- Визуализации прогрессии

## Как запустить

### Через Visual Studio 
1. Откройте файл `Lab_2.var26.sln`
2. Запустите программу

#### Пример использования
Пользователь вводит занчения первого элемента последовательности, разности элементов и количества элементов в соответствующие поля.
При неверном вводе подсчет результатов невозможен
При нажатии кнопки на листе SequnseGenerator происходит вывод арифметической прогрессии.
При нажатии кнопки на листе ProgressionSum происходит вывод суммы прогрессии.
При нажатии кнопки на листе ProgressionVisualizing происходит визуализация арифметической прогрессии.

##### Особенности программы
1. Полная валидация ввода (проверка на число, пустой ввод)
2. Графический интерфейс
3. Визуальное отображение результата


###### Требования 
1. .NET 6.0 или новее
2. Visual Studio 2022 (или любая среда с поддержкой C#)
